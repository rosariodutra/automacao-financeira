# 💰 Automação Financeira

Sistema de controle financeiro pessoal no terminal com lançamentos automáticos, categorização de despesas, análise de cenários e relatórios — desenvolvido em Python.


## ✨ Funcionalidades

- **Lançamentos** de receitas e despesas com categorização automática
- **Resumo financeiro** em tempo real com saldo atualizado
- **Análise por categoria** com gráfico de barras no terminal
- **Análise de cenários** — simula impacto de redução de despesas e aumento de receitas
- **Exportação de relatório** em `.txt`
- **Persistência em JSON** — dados salvos entre sessões

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/rosariodutra/automacao-financeira.git

# Entre na pasta
cd automacao-financeira

# Execute
python automacao_financeira.py
```

> Requer Python 3.6+. Sem dependências externas.
